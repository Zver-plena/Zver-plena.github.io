C. Paprika and Permutation

题意

Paprika loves permutations. She has an array 𝑎1,𝑎2,…,𝑎𝑛. She wants to make the array a permutation of integers 1 to 𝑛.

In order to achieve this goal, she can perform operations on the array. In each operation she can choose two integers 𝑖i (1≤𝑖≤𝑛) and x (x>0), then perform 𝑎𝑖 := 𝑎𝑖 mod 𝑥 (that is, replace ai by the remainder of 𝑎𝑖 divided by 𝑥). In different operations, the chosen 𝑖 and 𝑥 can be different.

Determine the minimum number of operations needed to make the array a permutation of integers 1 to 𝑛. If it is impossible, output -1.

A permutation is an array consisting of 𝑛n distinct integers from 1 to 𝑛 in arbitrary order. For example, [2,3,1,5,4] is a permutation, but [1,2,2] is not a permutation (2 appears twice in the array) and [1,3,4] is also not a permutation (n=3 but there is 4 in the array).

输入格式

Each test contains multiple test cases. The first line contains a single integer 𝑡$(1\le t \le10^4)$ — the number of test cases. Description of the test cases follows.

The first line of each test case contains an integer 𝑛 $(1\le n\le 10^5)$.

The second line of each test case contains 𝑛n integers 𝑎1,𝑎2,…,𝑎𝑛 $(1\le a_i\le 10^9)$.

It is guaranteed that the sum of 𝑛n over all test cases does not exceed $2\cdot 10^5 $.

输出格式

For each test case, output the minimum number of operations needed to make the array a permutation of integers 1 to 𝑛, or −1 if it is impossible.

Example

input
4
2
1 7
3
1 5 4
4
12345678 87654321 20211218 23571113
9
1 2 3 4 18 19 5 6 7
output
1
-1
4
2

思路

大致题意

对于给出有n个元素的数组,能否把他们变成1~n的形式,通过$a_i=a_i\mod k$的形式,其中k可以是任何数,求最少的操作次数

思路

  1. 首先一个数x的余数的范围为0~(x-1)/2,所以对于一个数越大,那么其余数的范围越大,所以对于每一个数应该尽可能的去匹配最大的未填入的数1~n

  2. 那么对于已经出现过的1~n的数,第一次出现我们就不用对他进行处理

  3. 然后就是对数组排序然后对数组的数和未填入的数的集合从大到小进行匹配,如果存在不能匹配的即未填入的数大于当前数的余数范围那么就是 -1,反之就是数组的大小为答案

    那么这样实现的时间复杂度为$O(nlogn)$

代码实现

/*
 * @Autor: Zver
 * @Date: 2021-12-20 17:30:16
 * @LastEditTime: 2021-12-20 22:19:29
 */
#include <algorithm>
#include <iostream>
#include <set>
#include <vector>
using namespace std;

int main()
{

    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        vector<int> v;
        set<int> s;
        for (int i = 1; i <= n; i++) {
            s.insert(i);
        }
        int number;
        for (int i = 0; i < n; i++) {
            cin >> number;
            if (s.find(number) != s.end())
                s.erase(number);
            else
                v.push_back(number);
        }
        sort(v.begin(), v.end());

        reverse(v.begin(), v.end());
        bool ok = false;
        for (auto x : v) {
            int count = (x - 1) >> 1;
            auto it = s.end();
            it--;
            if (count < *it) {
                ok = true;
                break;
            }
            s.erase(it);
        }
        if (ok) {
            cout << -1 << endl;

        } else {
            cout << v.size() << endl;
        }
    }
}
