树状数组

1.单点更新、区间查询

lowbit

int lowbit(int x) //算出 x 的最低位的 1 对应的值
{
    return x & (-x);
}

单点更新

代码实现
void update(int i, int x, int len) //在位置i加上x
{
    while (i <= len) {
        c[i] += x;
        i += lowbit(i);
    }
}

2.单点更新、单点查询

思路

构造和单点更新代码同上仅仅是传入的x不同
x传入的值为数组的差分值
这样区间求和就变成求单点的值

单点更新

代码实现
void update(int i, int x, int len) //在位置i加上x
{
    while (i <= len) {
        c[i] += x;
        i += lowbit(i);
    }
}

单点查询

代码实现
int getSum(int i) //求c[1,i]的和 == a[i]的值
{
    int sum = 0;
    while (i) {
        sum += c[i];
        i -= lowbit(i);
    }
    return sum;
}

3.区间更新、区间查询

思路

把x-y区间内的所有值全部加上k或者减去k，然后查询操作是问某个点的值
用两个树状数组来存
一个存储差分数组:$d1[i]$
一个存储:$d2[i] = d1[i]*(i-1)$

推导过程

$a_1+a_2+a_3···+a_n$
$=c_1+(c_1+c_2)+(c_1+c_2+c_3)····+(c_1+c_2+···c_n)$
$=nc_1+(n-1)c_2+(n-1)c_3····+c_n$
$=n(c_1+c_2+c_3+····c_n)-(0+c_2+2\cdot c_3+···(n-1)\cdot c_n)$
由此可以用d1和d2进行区间求和

区间更新

代码实现
// 将[l,r]的值全部加上x
//->updata(l,x,len);
//->updata(r,-x,len);
void updata(int i, int x, int len) //将i到len的值全部加上x
{
    int t = i;
    while (i <= len) {
        d1[i] += x;
        d2[i] += (t - 1) * x;
        i += lowbit(i);
    }
}

区间查询

求[1,i]的和

代码实现
int getSum(int i) //求[1,i]的和
{
    int sum = 0, cnt = i;
    while (i) {
        sum += cnt * d1[i] - d2[i];
        i -= lowbit(i);
    }
    return sum;
}
