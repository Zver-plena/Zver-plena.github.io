0.3.马克思主义的鲜明特征

1 . 科学性

马克思主义是在社会实践和科学发展的基础上产生的,并在自身发展过程中不断总结实践经验,吸取自然科学和社会科学发展的最新成就.

2. 人民性

人民至上是马克主义的政治立场,马克思主义政党把人民放在心中最高位置,一切奋斗都致力于实现最广大人民的根本利益.

3. 实践性

马克思主义从实践来,到实践中去,在实践中接受检验,并随实践不断发展的学说

4. 发展性

马克思主义是时代的产物,并随着时代,实践和科学的发展不断发展.

这些特征表示类类马克思主义的本质和使命,展现出马克思主义的理论形象.是科学性与革命性的统一

1.1.1.物质及其存在方式

哲学的物质范畴
  1. 古代朴素唯物主义用某一种或几种物质作为本原,来解释世界,在当时具有合理性和进步性
历史唯心主义

历史唯心主义有两个根本缺陷

  1. 看不到物质生产是人们思想动机的根源
  2. 不承认人民群众是推动发展的决定性力量

1.1.2.物质与意识的辩证关系

物质决定意识
  1. 意识在内容上是客观的,在形式上是主观的,是客观内容和主观形式的统一
  2. 观念的东西不外是移入人的头脑并在人的头脑中改造过的物质的东西而已
意识对物质具有能动作用

  • 意识对物质具有能动作用,意识的能动作用主要表现在

    1. 意识具有目的性和计划性

      • 马克思说过: 人在劳动过程结束时得到的结果,从这过程开始的时候就已经在劳动者的表象中存在着,即已经观念地存在着.

    2. 意识具有创造性

    3. 意识具有指导实践改造客观世界的作用

      列宁说过: 世界不会满足人,人决定以自己的行动来改变世界

    4. 意识具有调控人的行为和生理活动的作用

主观能动性和客观规律性的辩证统一

主观能动性 : 是指人们能动地认识世界,能动地改造世界的实际能力和作用

客观规律性 : 是指事物运动过程中本身所固有的本质的联系和必然的趋势

正确认识和把握物质与意识的辩证关系,还需要处理好主观能动性和客观规律性的关系

  1. 尊重客观规律性是正确发挥主观能动性的前提
  2. 只有充分发挥主观能动性,才能正确认识和利用客观规律.
  3. 正确发挥人的主观能动性,还要依赖一定的物质条件与物质手段—“巧妇难为无米之炊”,没有现实的原材料,人的意识再巧也创造不出任何物质的东西来

1.2. 事物的普遍联系和变化发展

1.2.1.联系和发展的普遍性

联系和发展的观点是唯物辩证法的总观点

联系的特点

  1. 联系具有客观性

  2. 联系具有普遍性

    1. 任何事物都不能孤立存在,都同其他事物处于一定的联系之中
    2. 整个世界是相互联系的统一整体

  3. 联系具有多样性

  4. 联系具有条件性

1.2.2.对立统一规律是事物发展的根本规律

对立统一规律是唯物辩证法的实质和核心

矛盾的普遍性

矛盾普遍性 : 是指矛盾存在于一切事物中,存在于一切事物发展过程中.

“矛盾无处不在,矛盾无时不有”就是对矛盾普遍性的形象表述

矛盾的特殊性

矛盾的特殊性: 是指是指各个具体事物的矛盾,每一个矛盾的各个方面在发展的不同阶段都有其特点

  1. 矛盾的特殊性决定了事物的不同性质

  2. 只有具体的分析矛盾的特殊性,才能认清矛盾的本质和发展规律,并采取正确的方法和措施去解决矛盾,推动事物的发展.

1.2.3.量变质变规律和否定之否定规律

量变质变规律

量变质变规律体现了事物发展的渐进性和飞跃性的统一

质 : 是指一事物区别于其他事物的内在规定性.

量 : 是指事物的规模,程度,速度等可以用数量关系表示的规定性.

  • 量变和质变的辩证关系

    1. 量变是质变的必要准备

      任何事情的变化都有一个量变的积累过程,没有量变的积累,质变就不会发生.

    2. 质变是量变的必然结果

      单纯的量变不会永远持续下去,量变达到一定程度必然引起质变

    3. 量变和质变相互渗透

1.2.4.联系与发展的基本环节

1.内容与形式

  • 内容

    1. 内容是指构成事物的一切要素的总和

  • 形式

    1. 形式是指把诸要素统一起来的结构或表现内容的方式

  • 任何事物都是内容与形式的统一

    1. 内容是事物存在的基础,对形式具有决定的作用

    2. 形式对内容具有反作用,适合的内容的形式,对内容的发展起积极的推动作用.

  • 形式具有相对独立性,同一内容可以用多种形式来呈现.

2. 本质与现象
3. 原因与结果
4. 必然与偶然
5. 现实与可能

1.3. 唯物辩证法是认识世界和改造世界的根本方法

1.3.1.唯物辩证法的本质特征和认识功能

唯物辩证法本质上是批判的和革命的

  • 恩格斯指出

    1. “马克思的整个世界观不是教义,而是方法,他提供的不是现成的教条,而是进一步研究的出发点和供这种研究的方法”

    2. “在辩证哲学面前,不存在任何最终的东西,绝对的东西,神圣的东西,他指出所有一切事物的暂时性.”

  • 马克思指出

    1. ·“辩证法在对现存事物的肯定的理解中同时包含对现存事物的否定的理解,即对现存事物的必然灭亡的理解”

    2. “辩证法对每一种现成形式都是从不断的运动中,因而也是从他的暂时性方面去理解”

    3. “辩证法不崇拜任何东西,任其本质来说,他是批判的和革命的”.

唯物辩证法是客观辩证法和主观辩证法的统一

  1. 在马克思主义哲学中,唯物论和辩证论是统一的

  2. 就唯物辩证法的本身来看,唯物论和辩证法的统一还体现为客观辩证法和主观辩证法的统一

    唯物辩证法即是客观辩证法,也是主观辩证法,是二者的有机统一.

知识点
  1. 唯物辩证法的总特征是
    • 联系和发展的观点

2. 实践与认识及其发展规律

2.1. 实践与认识

2.1.1. 实践的本质与基本结构

实践的本质

实践的定义 : 实践是人类能动地改造世界的社会性的物质活动

  • 实践的本质

    1. 实践具有客观存在性

      实践是人类改造世界的客观物质活动

    2. 实践具有直觉能动性

      人的实践活动是一种有意识的,有目的的活动

    3. 实践具有社会历史性

      实践是社会性的,历史性的活动.

实践的基本结构

  1. 实践主体

    实践主体:是指具有一定的主体能力从事现实社会实践活动的人

    • 实践主体有三种基本形态
      1. 个体主体
      2. 群体主体
      3. 人类主体

  2. 实践客体

    实践客体:是指实践活动所指向的对象

  3. 实践中介

    实践中介 : 是指各种形式的工具,手段以及运用这些工具手段的程序的方法.

实践形式的基本类型

  1. 物质生产实践

  2. 社会政治实践

  3. 科学文化实践

实践对认识的决定作用

index-材料分析

  • 辩证唯物主义认为:
    1. 在实践和认识之间,实践是认识的基础,实践在认识活动中起着决定性的作用.
    2. “实践的观点是辩证唯物论的认识论之第一和基本的观点“
  • 实现对认识的决定作用
    1. 实践是认识的来源
    2. 实践是认识发展的动力
    3. 实践是认识的目的
    4. 实践是检验认识真理性的唯一标准.

2.1.2. 认识的本质与过程

从实践到认识

感性认识:是指人们在实践的基础上,由感觉器官直接感受到的关于事物的现象,事物的外在联系,事物的各个方面的认识.

  • 感性认识的三种形式

    1. 感觉
    2. 知觉
    3. 表象

理性认识 : 是指人们借助抽象思维,在概括整理大量的感性材料的基础上,达到关于事物的本质,全体,内部联系和事物自身规律性的认识.

  • 理性认识的三种形式
    1. 概念
    2. 判断
    3. 推理
知识点
  1. 感性认识是对现象的认识,理性认识是对本质的认识
    1. ​ 认识的本质是主体对客体的能动反映,马克思主义认识论对认识的本质的概括
    2. 辩证唯物论的能动的反映论与旧唯物论的机械的反映论的对立
    3. 唯物论的反映论与唯心论的认识论对立
    4. 可知论与不可知论的对立

2.1.3. 实践与认识的辩证运动及其规律

index-材料分析

实践与认识的辩证运动,是一个由感性知识到理性认识,再由理性认识到实践的飞跃,是实践,认识,再实践,再认识,循环往复以至无穷的辩证发展过程.

这个过程是认识在实践的基础上沿着科学性方向不断深化发展的过程,也是实践在认识的指导下沿着合理性不断深入推进的过程.

2.2.3.真理与价值的辩证统一

价值的基本特性

  1. 价值的主体性

    价值的主体性: 是指价值直接与主体相联系,始终以主体为中心.

  2. 价值的客观性

    价值的客观性: 在一定条件下客体对于主体的意义并不依赖于主体的主观意识而存在.

  3. 价值的多维性

    价值的多维性: 是指每个主体的价值关系具有多样性,同一客体相对于主体的不同需要会产生不同的价值

    例如钻石对于主体的不同需要来说,会产生科学研究价值,审美价值,经济价值等.

  4. 价值的社会历史性

    价值的社会历史性 : 是指主体和客体的不断变化决定了价值的社会历史性.

价值评价及其特点

价值评价要以真理作为根据,要有利于人类主体的生存和发展,与社会历史的客观发展相一致,推动社会历史进步,以最广大人民的需要和利益为根本.

真理与价值的尺度

人们的实践活动总是受着真理尺度和价值尺度的制约

实践的真理尺度 : 是指实践中人们必须遵循正确反映客观事物本质和规律的真理

实践的价值尺度 : 是指在实践中人们都是按照自己的尺度和需要去认识世界和改造世界.

经济基础与上层建筑的矛盾运动及其规律

  • 经济基础

    • 经济基础 : 是指社会一定发展阶段的生产力所决定的生产关系的总和

  • 上层建筑 index-名词解释

    • 上层建筑 : 是指建立在一定经济基础之上的意识形态以及与之相适应的制度,组织和设施.
    • 自原始社会解体以来,上层建筑由两部分构成
      1. 意识形态
      2. 政治法律制度及设施组织

  1. 经济基础决定上层建筑,上层建筑反作用于经济基础,二者相互影响,相互作用.

  2. 任何社会的上层建筑都源于他的经济基础

  • 国家的实质

    • 国家是一个阶级统治另一个阶级的工具,是经济上占支配地位的阶级为维护其根本利益而建立起来的强制性的暴力机关,以保障其在政治也为统治阶级.

人民群众在创造历史的过程中的决定作用

5.1.1.资本主义由自由竞争到垄断

金融资本和 金融寡头

在金融资本形成的基础上,形成了金融寡头.

  • 金融资本: 是指由工业垄断资本和银行垄断资本混合在一起而形成的一种垄断资本

  • 金融寡头 : 是指操纵国民经济命脉,并在实际上控制国家政权的少数垄断资本家或垄断资本家集团.

    1. 他们支配大量的社会财富,控制整个国家的经济命脉和上层建筑,是垄断资本主义国家事实上的统治者(主宰者).
垄断利润和垄断价格

index-名词解释

  • 垄断利润 : 是指垄断资本家凭借其在社会生产和流通中的垄断地位而获得的超过平均利润的高额利润
知识点
  1. 垄断价格的形成和垄断利润的产生表明
    • 价值规律的作用形式发生了变化
  2. 主要资本主义国家相继自由竞争阶段发展到垄断阶段的时期是
    • 19世纪末20世纪初

国家垄断资本主义的形成及作用

国家垄断资本主义是指国家政权和私人垄断资本融合在一起的垄断资本主义

  • 国家垄断资本主义的产生,是垄断资本主义生产关系在自身范围内的部分质变,标志着资本主义发展进入了新的阶段
垄断资本向世界范围的发展

输出资本的主体来看,主要有两种

  1. 私人资本输出
  • 私人资本输出是为了获得高额垄断利润或利息,同时也是加强剥削和控制发展中国家的经济手段
  1. 国家资本输出
  • 一方面为私人垄断资本服务
  • 一方面为维护资本主义世界体系服务
